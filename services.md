# User & Shooter

## User
If user didnt register before, they would have to register an user account.
## Shooter
There are two types of shooter records:
1. The shooter that does not have a registered user account.
2. The shooter that has a registered user account.
### Shooter without user account
This type of shooter are created by the team admin which is for the the team members to use this system without them to register. After the team member who doesn't register get registered, they can adopt the shooter and this shooter will turns into a registered shooter (type 2).

### Shooter with user account
This type of shooter are created by the user itself.

# Team
## Creation of Team
The user can form a team, the user that create the team have full control over the team. which can be edit, delete, add members, etc.. most importantly, the owner can grant or revoke the admin role to the team members.

## Shooters
All shooters are listed in the team page. The team admin can add new shooters invite the team. if the team member doesn't have a user account. they can create a shooter. ([ref](#shooter-without-user-account))

# Stage
## Creation of Stage
All of the user can create a stage, and assign the deisgner (default is the user itself) to the stage. The stage can be used in all the matches.

# Match
## Match admin
The user can form a group of match admin, which contains serval user who have the permission to manage the match.
